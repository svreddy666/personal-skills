# Personal Skills

A personal repository to track, learn, and practice the skills that matter to you.

## Purpose
- Collect skill definitions, goals, step-by-step learning plans, and resources.
- Track progress and tasks for each skill.

## Structure
- [skills/README.md](skills/README.md) — guidance for adding and managing skills.
- [skills/skill-template.md](skills/skill-template.md) — template for creating new skill entries.
- CONTRIBUTING.md — contribution and PR guidelines.
- LICENSE — repository license.
